# 夏令营项目验收清单

## 通用要求

- [ ] 不是只运行官方 demo，而是完成训练、复现或改造。
- [ ] 能保存最终 checkpoint。
- [ ] 能加载 checkpoint 做测试。
- [ ] 有训练曲线。
- [ ] 有仿真 demo 视频。
- [ ] README 写清楚环境配置、运行命令和结果路径。

## Locomotion

- [ ] 机器人稳定站立 / 移动。
- [ ] 能跟踪不同速度指令。
- [ ] 至少 3 组速度指令测试。
- [ ] 复杂地形由 IsaacLab terrain 生成，若选择四足或轮足。
- [ ] 连续运动过程中不频繁摔倒。

## Navigation

- [ ] SEA-Nav 环境或自建导航环境能运行。
- [ ] 能训练或复现实验流程。
- [ ] 能从起点到目标点。
- [ ] 记录成功率、碰撞次数、导航轨迹。
- [ ] 动态障碍物任务至少包含 3 个动态障碍物。

## Manipulation

- [ ] Franka Reach baseline 能训练。
- [ ] 机械臂末端能稳定接近随机目标点。
- [ ] 按按钮任务记录末端误差和按钮触发成功率。
- [ ] 提袋子任务记录提起高度、保持时间和成功率。
